# Why groceries got so expensive in Canada (2000-2025)

#Project Overview
This project anayzes why grocery prices in Canada increased significantly between 2000 and 2025

Using open data from Stats Canada, the project investigates the relationship between:

- Consumer Price Index
- Agricultural product prices
- Average weekly wages
- Grocery and foood inflation

The goal is to understand how food affordability has changed over time and identify major drivers of grocery price increases.
